Features

- Save and organize code snippets.
- Edit and delete existing snippets.
- Export snippets to a ZIP file.

Load the Extension in Chrome

1. Open the Chrome browser.
2. Navigate to chrome://extensions/ in the address bar.
3. Enable "Developer mode" by toggling the switch in the top-right corner.
4. Click on the "Load unpacked" button.
5. Navigate to the folder where you extracted the SnippetSaver extension.
6. Select the folder and click "Open".

The SnippetSaver extension should now be installed and visible in your list of extensions. You can pin it to your browser toolbar for easy access.

Usage
Click on the SnippetSaver icon in your browser toolbar to open the extension.
Save new snippets by entering the snippet name and content, then clicking "Save snippet".
Access and manage your saved snippets by clicking "Load snippets".
